This lists from and uploads to your chosen prefix. From your screenshot, your file is in geophoto/scenarios/, so choose scenarios (flat root).
Uploads blocked by Row Level Security.
In Supabase SQL editor, run policies like: 
-- Allow public read of geophoto
create policy "geophoto read anon" on storage.objects
for select to anon using (bucket_id = 'geophoto');

-- Allow public insert (uploads) to geophoto
create policy "geophoto insert anon" on storage.objects
for insert to anon with check (bucket_id = 'geophoto');
(Optional, tighter): limit uploads to the scenarios/ prefix: 
create policy "geophoto insert anon scenarios only" on storage.objects
for insert to anon with check (bucket_id = 'geophoto' and position('scenarios/' in name) = 1);
